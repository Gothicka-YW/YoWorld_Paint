# YoWorld Paint Tutorials (YouTube Script Draft)

Use this as a starting script pack for short tutorial videos. Each section is written so you can read it naturally on camera and trim as needed.

## Video 1: Home Tab - Quick Image Upload + Set Paint Board

### Title
YoWorld Paint Set Up & Home Tab Tutorial | YoWorld Paint Boards Made Easy

### Hook (0:00-0:10)
"In this video, I will show you how to set up the YoWorld Paint extension and use the Home tab to upload an image on to YoWorld paint boards. The process is quick and easy because I like to be as lazy and efficient as possible! The link to the download is found in the description of this video."

### Script (0:10-2:30)
"In your browser - it can be Google Chrome, Edge, Opera, Brave, or any other Chromium based browser - I'm using Google Chrome. Go to the Chrome Web Store and search for YoWorld Paint. Click "Add to Chrome." Now go to this puzzle piece at the right of your address bar to pin the icon so you can find it easily. 

"Open YoWorld Paint and stay on the Home tab. At the top, you will see Enable Redirect. Keep this off for now."

"You have two ways to set an image. Method one: paste a direct image URL into the field and click Set New Image."

"Method two, and usually faster: use Quick Image Uploader. You can click to choose a file, drag and drop an image, or paste directly from your clipboard."

"Choose your host, then click Upload. If Auto-set as Current Image is checked, the extension will automatically use that uploaded image in the Home tab field."

"Now turn on Enable Redirect, go into YoWorld, save your paint board, then come back and turn Redirect off when you are done."

### Outro (2:30-2:45)
"That is the full Home tab flow. In the next video, we will build a 3 by 2 Sales Board from YoWorld Info."

## Video 2: Sales Boards Tab - Build a 3x2 Board from YoWorld Info

### Title
YoWorld Paint Sales Boards Tutorial | Capture, Preview, Export

### Hook (0:00-0:10)
"Let us build a clean 3x2 sales board from yoworld.info using the Sales Boards tab."

### Script (0:10-3:00)
"Open the Sales Boards tab, then click the YoWorld Info link to open yoworld.info/template."

"Back in the extension, click Pick Card Selector, then click the first card image you want in your board."

"Use Reset Selectors if you picked the wrong area."

"Click Preview 3x2 to generate your board. You can click Restore List if you need to reload your last captured sequence."

"When it looks right, click Export Crop to save the final 390 by 260 board image."

"Then upload that image in the Home tab using Quick Upload or manual link paste."

### Outro (3:00-3:15)
"That is the full Sales Boards workflow. Next up is Transform for perspective fixes and cleanup."

## Video 3: Transform Tab - Perspective Fix and Export

### Title
YoWorld Paint Transform Tutorial | Perspective Fix for Better Boards

### Hook (0:00-0:12)

### Script (0:12-3:30)

"You can start with a preset, then fine-tune matrix values if needed."

"Click Apply to run the transform, then review the preview canvas carefully."

## Video 7: Tips and Tricks - Shortcuts, Snipping, and Clipboard Workflow

### Title
YoWorld Paint Tips and Tricks | Speed Up Your Workflow (Windows + Mac)

### Hook (0:00-0:12)
"Here are the best speed shortcuts for YoWorld Paint, including screenshot keys, clipboard upload, and quick navigation."

### Script (0:12-4:00)
"Tip one: capture only what you need using built-in snipping shortcuts."

"On Windows, press Shift + Windows key + S to open Snipping Tool selection mode."

"On Mac, press Command + Shift + 4 to capture a selected area, or Command + Shift + 5 for full screenshot controls."

"Tip two: skip saving files when possible. Copy and paste directly into YoWorld Paint."

"Windows: copy with Ctrl + C and paste with Ctrl + V."

"Mac: copy with Command + C and paste with Command + V."

"In the Home tab Quick Image Uploader area, pasted images can be uploaded directly, which is much faster than saving every image first."

"You can still save and upload the normal way if you prefer that workflow."

"Tip three: fast tab and window control while you gather references."

"Windows: Ctrl + T for new tab, Ctrl + W to close tab, Ctrl + Tab to switch tabs."

"Mac: Command + T for new tab, Command + W to close tab, Control + Tab to switch tabs in many browsers."

"Tip four: keep source and extension side by side. Use browser split view or side panel mode while you reference yoworld.info or generator output."

"Tip five: use consistent naming when exporting boards and tiles so you can find files quickly later."

### Outro (4:00-4:15)
"Use these shortcuts daily and your board creation time drops a lot. If you want, I can make a printable shortcut card next."
