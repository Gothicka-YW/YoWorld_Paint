# YoWorld Paint Installation Guide

Install from the Chrome Web Store, then pin the icon so it is always one click away.

---

## Step-by-Step (Chrome Web Store)

1. Open **Google Chrome**.

2. Open the **Chrome Web Store**.
   - Option A: Enter `https://chromewebstore.google.com` in the address bar.
   - Option B: Search for **Chrome Web Store** in Google and open the official result.

3. In the Web Store search bar, type **YoWorld Paint**.

4. Open the **YoWorld Paint** extension listing.

5. Click **Add to Chrome**.

6. In the permissions prompt, click **Add extension**.

7. Wait for the install confirmation.
   - You should see a message that **YoWorld Paint was added**.

---

## Pin the Extension Icon (Recommended)

1. Click the puzzle icon (**Extensions**) in the top-right of Chrome.
2. Find **YoWorld Paint** in the extension list.
3. Click the pin icon next to **YoWorld Paint**.
4. The YoWorld Paint icon will remain visible in your toolbar.

---

## Launch

Click the pinned **YoWorld Paint** icon to open the extension.

---

## Quick Troubleshooting

- If you do not see **Add to Chrome**, make sure you are on the official extension listing page.
- If the icon is missing after install, open the **Extensions** menu and pin it manually.

---

## Installing in Microsoft Edge and Opera

If you are not using Chrome, the steps are slightly different:

### Microsoft Edge

1. Open **Microsoft Edge**.
2. Go to the **Chrome Web Store** listing for YoWorld Paint.
3. If prompted, enable **Allow extensions from other stores** in Edge.
4. Click **Add to Chrome** (or **Get**) and confirm installation.
5. Open `edge://extensions` or the Extensions menu to pin **YoWorld Paint** to the toolbar.

### Opera

1. Open **Opera**.
2. Install the Opera add-on **Install Chrome Extensions** (one-time helper).
3. Then open the **Chrome Web Store** listing for YoWorld Paint.
4. Click **Add to Opera** / **Add to Chrome** and confirm.
5. Open Opera's Extensions manager to enable and pin YoWorld Paint.

Note: UI labels may vary slightly by browser version, but the install flow is the same: allow third-party store extensions, install, then pin.
